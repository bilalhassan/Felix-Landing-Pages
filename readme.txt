Build your marketing Landing Page or Flyer in minutes. Easy to use page builder with WordPress Customizer

== Description ==

This plugin allows you to easily create and build a landing page without having to work with any code. You can use the Customizer to show/hide sections, as well as re-order them. You can use 
Felix Landing Page to create a marketing or advertising page on your site for a product or a service. Felix integrates with WooCommerce, and allows you to select which products you want 
to display. You can use this to build a store flyer, or any other kind of page. 

Felix works with any theme, allowing you to build your landing page with a unique design. Felix allows you to set the font and color of your template.

Plugin Features
- Page builder
- Hide/re-order sections
- Multiple Call-to-action sections
- WooCommerce integration
- Live Customizer
- Appearance/font/color customization
- Hero/Jumbotron featured image area
- Navigation links
- Articles section
